# Huobao Drama 前端使用教程

本教程只讲前端页面如何使用，按实际操作顺序写。

## 1. 进入剧集工作流

1. 打开项目首页，进入某个剧本。
2. 进入目标集数（如第1集）的工作流页面。
3. 顶部步骤一般是：
- 章节内容
- 角色/场景与图片
- 分镜拆解

## 2. 分镜前的必要操作

1. 在“章节内容”中填写并保存剧本文本。
2. 完成角色和场景提取（如页面有按钮则先执行）。
3. 需要时生成角色图/场景图。
4. 点击“AI自动拆分”生成分镜。

分镜成功后会自动进入“专业编辑器”。

## 3. 专业编辑器页面结构

页面分三栏：
- 左侧：镜头列表（脚本结构）
- 中间：时间线与素材区
- 右侧：镜头属性 / 镜头图片 / 视频生成 / 音效与配乐 / 视频合成

## 4. 左侧镜头列表怎么用

1. 点击某个镜头：切换当前镜头。
2. 点击“添加”：新增镜头。
3. 点击镜头右侧删除图标：删除镜头。

建议先把镜头顺序和时长确认好，再批量生成视频。

## 5. 右侧“镜头属性”怎么用

按这个顺序改最稳：

1. 选择场景（Scene）
- 点“选择场景”，绑定当前镜头背景。

2. 添加角色（Cast）
- 点“添加角色”加入角色。
- 点角色上的关闭图标可移除。

3. 添加道具（Props）
- 点“添加道具”加入道具。
- 点关闭图标可移除。

4. 视觉设置
- 景别
- 运镜方式
- 镜头角度

5. 文本内容
- 动作描述
- 动作结果
- 对白
- 镜头描述
- 音效
- 配乐提示
- 氛围

保存规则：
- 下拉框改动会自动保存。
- 文本框失焦（点击到别处）会自动保存。

## 6. 右侧“镜头图片”怎么用

目标：生成后续视频所需的参考图。

操作步骤：
1. 选择帧类型：首帧 / 尾帧 / 动作序列 / 关键帧。
2. 点“提取提示词”。
3. 视情况修改提示词文本。
4. 点“生成图片”。
5. 在结果区挑图，不满意就删除重生。
6. 也可以点“上传图片”直接上传参考图。

提示：
- 动作序列适合连续动作镜头。
- 首帧和尾帧适合做镜头衔接。

## 7. 右侧“视频生成”怎么用

1. 选视频模型。
2. 选参考图模式（按模型支持项来选）。
3. 设置时长（通常跟镜头时长接近）。
4. 选择参考图（可选首帧/尾帧/关键帧/动作序列图）。
5. 点击“生成视频”。
6. 生成成功后点击“加入素材库”。

提示：
- 同一镜头可多次生成，最后选最佳版本入库。
- 镜头衔接不顺时，优先使用“上一镜头尾帧 + 当前镜头首帧”。

## 8. 中间时间线怎么用

1. 从素材区把视频放入时间线轨道。
2. 调整顺序与位置。
3. 检查每段时长与过渡是否自然。
4. 点击合成相关按钮发起整集合成。

## 9. 右侧“视频合成”怎么看

合成任务状态一般有：
- pending
- processing
- completed
- failed

可执行：
- completed：预览/下载
- failed：查看报错后回到镜头或素材修正再合成

## 10. 建议的实际工作顺序

1. 先修“镜头属性”（剧情和视觉语义）
2. 再做“镜头图片”（参考图质量）
3. 再做“视频生成”（逐镜头产出）
4. 最后做“时间线 + 视频合成”（整片输出）

这样返工成本最低。
